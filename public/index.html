Website Down Tester

This tool helps you check if a website is down for everyone or just you. Simply enter the URL and simulate a limited number of requests (max 100) to monitor uptime.

How It Works

  • Enter the full URL of the website you want to test.
  • Choose the number of requests (up to 100).
  • Click "Start" to send the requests and view progress.
